## Getting Started

Follow these steps to run the project locally:

```bash
# Clone this repository
git clone https://github.com/axshek/Bouncy-Documents.git
cd <Bouncy-Documents>

# Install dependencies
npm install
# or
yarn

# Start development server
npm start
# or
yarn start

# Build for production
npm run build
# or
yarn build
